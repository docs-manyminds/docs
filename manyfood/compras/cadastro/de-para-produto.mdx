# De Para de Produtos

**Menu:** B1Food → Documentos Fiscais → De Para de Produtos

---

O **De Para de Produtos** é utilizado para criar o **DE PARA** do **código do SAP** Business One com o código do produto do seu **fornecedor**, por exemplo:

O Código do seu item (ÁGUA COM GÁS) no **SAP B1 é 101001**, e quando for comprar esse produto do **Fornecedor A** o código dele é **12345** e quando for comprar esse mesmo produto do **Fornecedor B**, o código dele é **654321**, portanto esse **De Para de Produtos** serve para o B1Food identificar qual é o produto do SAP quando comprar esse produto do **Fornecedor A** e quando comprar do **Fornecedor B**

<Frame>
<img src="/images/doc_fiscal_de_para_01.png" alt="Tela inicial do de-para de produtos" />
</Frame>

<Warning>
Geralmente ele é criado no momento do lançamento, pois o B1Food irá mostrar qual produto falta de para
</Warning>

Nessa listagem irá aparecer todos que já foram criados, mas é possível criar novos **De Para de Produtos** ou até mesmo realizar edições nos cadastros já realizados

Para criar um novo **De Para de Produtos** (caso não for realizar pelo processo de **Nota Fiscal de Entrada**), basta clicar no botão **Novo**

<Frame>
<img src="/images/doc_fiscal_de_para_05.png" alt="Botão novo de-para" />
</Frame>

Nessa tela de cadastro, é necessário realizar todo preenchimento dos campos e depois clicar no botão **Salvar**

<Frame>
<img src="/images/doc_fiscal_de_para_06.png" alt="Tela de cadastro de de-para" />
</Frame>

<Tabs>
<Tab title="Campos">
- **Fornecedor**: Informar o fornecedor que está fazendo o **De Para de Produtos**
- **ID Origem**: Informar o código do item do seu fornecedor
- **Produto**: Informar o item que irá entrar no SAP caso realize a compra desse fornecedor com o código informado
- **Unidade Medida**: Informar qual unidade de medida que geralmente realiza compra desse item do fornecedor
</Tab>
</Tabs>

Para realizar edições de um registro já criado, é possível modificar o cadastro ou até realizar exclusão do mesmo

<Frame>
<img src="/images/doc_fiscal_de_para_02.png" alt="Ações disponíveis para o de-para" />
</Frame>

<Tabs>
<Tab title="✏️ Editar">
A ação **Editar** irá abrir o **De Para de Produtos** para realizar alguma edição, portanto irá carregar os dados do registro que deseja editar, basta realizar alterações que deseja e clicar em **Salvar**
<Frame>
<img src="/images/doc_fiscal_de_para_04.png" alt="Edição de de-para" />
</Frame>
</Tab>

<Tab title="❌ Excluir">
A ação **Excluir** irá apagar o **De Para de Produtos** criado
<Frame>
<img src="/images/doc_fiscal_de_para_03.png" alt="Exclusão de de-para" />
</Frame>
</Tab>
</Tabs>
